## sed
```shell
sed -i "s/模式/替换(/g" `grep "模式" -rl  --exclude-dir=排除目录 路径`
举例:
    sed -i "s/hello/world/g" `grep "hello" -rl  --exclude-dir=.svn .`   <=> 在当前目录下(排除.svn目录)将所有hello替换成world
```

## tcpdump
```
tcpdump tcp -i 网卡 -X port 端口
tcpdump -i 网卡 -X port [src|dst] host 主机名 and 端口
```
